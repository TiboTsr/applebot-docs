❓ Guide des Commandes

Toutes les commandes du bot sont des Commandes Slash (/).

← Retour à l'accueil

Fonctionnalités Principales (Publiques)

/latest [os]

Affiche la dernière version de firmware enregistrée dans la base de données pour un système d'exploitation Apple spécifique.

Paramètre : os (Choix parmi iOS, macOS, iPadOS, watchOS, tvOS, audioOS, visionOS)

Exemple : /latest os: iOS
Résultat : Affiche l'Embed de la dernière version d'iOS (ex: iOS 17.5.1).

/compat [version]

Recherche une version ou un Build ID spécifique dans la base de données pour lister les appareils compatibles.

Paramètre : version (ex: 17.5.1 ou 21F90)

Exemple : /compat version: 21F90
Résultat : Affiche un Embed détaillé listant tous les appareils qui reçoivent ce Build ID.

/maj [recherche]

La commande de recherche la plus complète. Elle cherche par version, Build ID ou nom d'appareil (ex: 'iPhone 15 Pro Max') dans toute la base de données (limitée aux 10 meilleurs résultats).

Paramètre : recherche (texte libre)

Exemple 1 : /maj recherche: iPhone 15 Pro
Exemple 2 : /maj recherche: 17.4
Résultat : Affiche un ou plusieurs Embeds correspondants à votre requête.

/ping

Vérifie la latence du bot (vitesse de réponse) entre Discord et le serveur.

Exemple : /ping
Résultat : Affiche la latence en millisecondes (ex: "Ma latence est de 45.21 ms.").

/help

Affiche un résumé des commandes directement dans Discord.

Commandes Administrateur (Propriétaire du bot)

/forcecheck

Force le bot à lancer immédiatement une vérification des mises à jour auprès d'AppleDB, sans attendre le prochain intervalle programmé.

/testembed

Envoie un embed de notification simulé dans le canal de notifications configuré (pour vérifier le formatage et les permissions).
